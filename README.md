Pure frontend -  clean, responsive landing page for a freelance automation & web development business.

## Features
- Fully responsive (mobile, tablet, desktop)
- Sticky navigation bar
- Hero section with code snippet visual
- Stats bar
- Services cards with hover effects
- Call-to-action section
- Pure HTML + CSS — no frameworks, no dependencies

## Usage
Just open `index.html` in any browser. No setup needed.
